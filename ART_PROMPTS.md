# VBookReader Story Art Prompts

> Goal: replace all Unsplash placeholder URLs in `stories.js` with original, consistent, child-friendly illustrations.
> Style target: soft watercolor/digital picture-book style, gentle lighting, rounded shapes, expressive but simple faces, no photorealism, no extra fingers, no text.
> Source strategy: generate with free AI image tools (Microsoft Designer, Leonardo.ai, Playground AI, Ideogram, Pixabay AI) or photograph your own drawings.


## File-naming convention

Save each image with this pattern so `stories.js` stays easy to update:

```
assets/{story-id}/cover.jpg
assets/{story-id}/page-01.jpg
assets/{story-id}/page-02.jpg
assets/{story-id}/page-03.jpg
assets/{story-id}/page-04.jpg   (only for stories with 4 pages)
```

Examples:
- `assets/rusty/cover.jpg`
- `assets/library/page-03.jpg`
- `assets/wavelength/page-04.jpg`

Use `.webp` or `.jpg`, width 1200 px, moderate compression. Keep file sizes under 200 KB each for fast loading.
